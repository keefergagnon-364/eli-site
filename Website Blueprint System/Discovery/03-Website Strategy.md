# 03 - Website Strategy

## Purpose

The Website Strategy document is the primary source of truth for the project.

All content generation, reviews, design decisions, SEO recommendations, and website development should reference this document.

The strategy combines information from:

* Discovery Questionnaire
* Website Audit
* Competitor Analysis
* Missing Asset Reports

The goal is to create a complete blueprint before any website content or code is generated.
