# 02 - Website Audit

## Purpose

The Website Audit evaluates the client's current online presence and identifies opportunities for improvement.

The audit combines:

* Human Review
* AI Analysis
* Competitor Research

The goal is to determine:

* Current strengths
* Current weaknesses
* Competitive opportunities
* Missing trust elements
* SEO opportunities
* Conversion opportunities

This audit feeds directly into the Website Strategy document.
