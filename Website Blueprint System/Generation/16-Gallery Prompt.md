# 16 - Gallery Prompt

## Purpose

Generate the Gallery Strategy and Gallery Section for the website.

The Gallery should provide visual proof of the company's work.

Visitors should leave the Gallery believing:

* The company produces quality work
* The company has real project experience
* The company can successfully complete similar projects

The Gallery is a proof section first and a design element second.

# Gallery Priorities

Ranked in order of importance:

1. Showcase Quality
2. Improve SEO
3. Generate Leads
4. Visual Appeal
5. Build Trust
6. Show Services

---

# Gallery Philosophy

The Gallery should:

Show Real Work

before

Describe Work

Visual proof is more persuasive than marketing claims.

Whenever possible:

Use real project photos.

Avoid excessive stock photography.

# Caption Strategy

Generate recommendations for image captions.

Captions should:

* Explain the project
* Reinforce trust
* Support SEO
* Provide context

Example:

Tree Removal - Show Low, AZ

Removed a hazardous pine tree near a residential property while protecting surrounding landscaping.

---

# SEO Rules

When appropriate:

Include:

* Service Keywords
* Service Area Keywords
* Project Type Keywords

Support local SEO goals.

Do not keyword stuff.

# Failure Conditions

The Gallery fails review if:

* Real project photos are ignored
* Missing assets are not reported
* SEO opportunities are ignored
* Captions are not recommended
* Gallery serves only visual purposes
* Trust opportunities are ignored
* Stock photos are recommended before better alternatives

Revise until all failure conditions are resolved.
