# 22 - Visual Design System Prompt

## Purpose

Generate the complete Visual Design System for the website.

The Visual Design System establishes a consistent visual language that supports:

* Professional appearance
* Trustworthiness
* Readability
* Brand personality
* Accessibility
* Conversion

The goal is not simply to create an attractive website.

The goal is to create a design system that reinforces the client's brand while improving user experience and lead generation.

# Design Priorities

Ranked in order of importance:

1. Readability
2. Accessibility
3. Trustworthiness
4. Professional Appearance
5. Brand Personality
6. Originality
7. Speed
8. Modern Design

Design should support usability before aesthetics.

---

# Design Philosophy

Prioritize:

Clear Communication

before

Visual Decoration

Every visual decision should improve:

* Readability
* Trust
* Navigation
* Conversion

Avoid unnecessary visual effects.

# Spacing & Layout Guidelines

Define recommendations for:

* Section spacing
* Content width
* Grid system
* Card spacing

---

# Animation Strategy

Animation level:

Minimal

Recommended:

* Fade-ins
* Smooth scrolling
* Hover effects
* Card elevation
* Button transitions

Avoid:

* Flashy animations
* Auto-playing effects
* Distracting motion

Every animation should serve a purpose.

---

# Accessibility Review

Evaluate the design for:

* Color contrast
* Readability
* Keyboard navigation support
* Focus states
* Touch target sizing
* Responsive typography

Recommend improvements where needed.

---

# Performance Considerations

Recommend design choices that support:

* Fast loading
* Optimized images
* Lightweight icons
* Minimal animation
* Responsive performance

Avoid visual choices that significantly reduce page speed.

---

# Visual Consistency Review

Ensure consistency across:

* Colors
* Typography
* Buttons
* Forms
* Cards
* Icons
* Spacing
* Imagery
* Animations

The website should feel unified from page to page.

---

# Failure Conditions

The Visual Design System fails review if:

* Readability is compromised
* Accessibility is ignored
* Visual style conflicts with the brand
* Components are inconsistent
* Animations distract from content
* Existing branding is discarded without justification
* Design choices negatively impact performance
* The system does not support trust or conversion

Revise until all failure conditions are resolved.
