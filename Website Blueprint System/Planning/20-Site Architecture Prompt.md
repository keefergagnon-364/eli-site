# 20 - Site Architecture Prompt

## Purpose

Generate the website architecture blueprint.

The Site Architecture defines:

* Website Structure
* Navigation Structure
* User Flow
* Conversion Paths
* SEO Structure
* Mobile Experience

All content generation and website development should follow the architecture blueprint.

The goal is to create a website that is:

* Easy to use
* Easy to navigate
* Easy to convert
* Easy to expand

# Architecture Priorities

Ranked in order of importance:

1. User Experience
2. Lead Generation
3. SEO
4. Mobile Experience
5. Simplicity
6. Future Scalability

---

# Architecture Philosophy

The website should:

Guide Users

before

Impress Users

A beautiful website that does not convert is a failure.

Every page and section should support:

* User Understanding
* Trust Building
* Lead Generation

* Unclear labels

# Failure Conditions

The Architecture fails review if:

* Navigation is confusing
* Conversion paths are unclear
* Mobile users are ignored
* SEO opportunities are ignored
* Service page opportunities are ignored
* Internal linking is ignored
* Future growth is ignored
* Architecture does not support lead generation

Revise until all failure conditions are resolved.
