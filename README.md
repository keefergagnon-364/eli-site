Client Workflow
Step 1: Log Into Jobber

Client signs into their Jobber account.

Step 2: Enable Online Booking / Request Form

Depending on their Jobber plan and current interface, they'll navigate to something similar to:

Settings
  → Client Hub
  → Request Form

or

Settings
  → Online Booking

Jobber occasionally changes menu names, but it's usually under Client Hub, Online Booking, or Request Forms.

Step 3: Configure the Form

They'll typically choose:

Services
✓ Landscaping
✓ Irrigation
✓ Lawn Care
✓ Tree Trimming
Customer Information
✓ Name
✓ Phone
✓ Email
✓ Address
Request Details
✓ Description
✓ Preferred Date
Step 4: Publish the Form

Jobber generates a public URL.

It will look something like:

https://clienthub.getjobber.com/client_hubs/xxxxx/request_work

or

https://secure.getjobber.com/request/xxxxx

The exact URL format may vary.
